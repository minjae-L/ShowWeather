# 날씨알려줘-iOS

## 기능정의
- 원하는 지역의 주소 또는 건물명을 검색할 수 있습니다.
- 해당 지역의 날씨를 알 수 있습니다.
- 지역 날씨를 저장하고 한 눈에 볼 수 있습니다.
